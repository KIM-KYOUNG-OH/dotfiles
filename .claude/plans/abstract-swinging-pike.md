# Claude Code 통합 사용법 문서화 계획

## 현황 분석

### 문제점
현재 vault-intelligence 문서는 **CLI 직접 실행** 위주로 작성되어 있습니다:
- USER_GUIDE.md: 1,672라인의 CLI 명령어 매뉴얼
- EXAMPLES.md: CLI 배치 스크립트 위주

**실제 사용 패턴**과 괴리:
- 사용자는 Claude Code에서 자연어로 요청
- 문서는 CLI 직접 실행만 안내

### 이미 반영된 부분
- CLAUDE.md: Claude가 참조하는 지침 문서 (수정 불필요)
  - "사용자의 요청을 처리하기 위해 검색이 필요한 경우 vault intelligence 시스템을 이용"
- EXAMPLES.md 실제 활용 사례 섹션: AI Practice, AI 한계 MOC, 브런치 글 작성
