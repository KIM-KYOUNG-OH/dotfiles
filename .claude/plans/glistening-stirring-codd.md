# 1on1 미팅 문서 정리 계획

## 작업 개요
기획팀 유선님과의 첫 1on1 미팅 내용을 Obsidian 문서로 정리

## 입력 자료
- 사용자의 메모 (핵심 키워드)
- 녹취록 (상세 대화 내용)

## 출력 파일
- 위치: `notes/dailies/2026-01-기획팀-1on1.md`
- 형식: Obsidian 1on1 미팅 노트

## 문서 구조
1. **Frontmatter**: 날짜, 참석자, 태그
2. **참석자 정보**: 유선님 배경 및 경력
3. **현재 업무**: CS/SR 지원, 이벤트 페이지 개선
4. **유선님의 목표와 강점**
   - UX/디자인 배경 강점
   - 데이터 분석 역량 개발 희망
   - PM으로 성장 목표
5. **멘토링 포인트** (백명석님 조언)
   - 시간과 배움의 가치
   - 좋은 롤모델의 중요성
   - 장기적 성장 관점
   - 결정장애는 신중함의 장점
6. **Action Items / 후속 조치**

## 작업 단계
1. 녹취록에서 핵심 내용 추출
2. 1on1 문서 템플릿에 맞게 구조화
3. hierarchical tag 적용
4. 파일 생성
